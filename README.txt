# Johnny & Coco's Magical Journey

一個充滿愛意的生日冒險遊戲！

## 遊戲特色

- 經典平台跳躍玩法
- 收集愛心提升分數
- 回答問題獲得獎勵
- 多種結局等待解鎖

## 經典 RPG 游標系統

遊戲現在包含完整的經典 RPG 風格游標效果，讓選單體驗更加豐富！

### 基本游標效果

- **抖動 (Shake)**: 選中選項會左右小幅度搖擺
- **脈衝 (Pulse)**: 選中選項會有輕微的縮放效果
- **發光 (Glow)**: 選中選項會有發光效果
- **符號閃爍**: `>` 和 `<` 符號會慢慢閃爍，吸引注意力
- **顏色區分**: 選中選項呈現白色，與未選中選項形成對比
- **統一效果**: 所有選單選項（開始菜單、COCO對話、問題選項）都使用相同的游標效果
- **一致樣式**: 所有選項都使用純文字樣式，沒有按鈕框框，保持視覺一致性

### 游標樣式主題

#### 1. 經典游標
```css
.rpg-cursor-classic
```
- 傳統 RPG 風格的游標
- 適中的抖動頻率
- 白色選中效果

#### 2. 魔法游標
```css
.rpg-cursor-magical
```
- 帶有發光效果的游標
- 較慢的抖動
- 白色選中效果

#### 3. 傳奇游標
```css
.rpg-cursor-legendary
```
- 高級的游標樣式
- 包含脈衝效果的組合動畫
- 白色選中效果

### 游標狀態

- **菜單游標** (`.rpg-cursor-menu`): 白色，適合主菜單
- **戰鬥游標** (`.rpg-cursor-battle`): 白色，適合戰鬥場景
- **物品游標** (`.rpg-cursor-inventory`): 白色，適合物品欄

### 使用方法

在 JavaScript 中，為選中的選項添加相應的 CSS 類：

```javascript
// 基本游標
<span class="option-selected">選項文字</span>

// 經典游標
<span class="option-selected rpg-cursor-classic">選項文字</span>

// 魔法游標
<span class="option-selected rpg-cursor-magical">選項文字</span>

// 傳奇游標
<span class="option-selected rpg-cursor-legendary">選項文字</span>
```

## 控制方式

- **方向鍵**: 移動角色
- **空格鍵**: 跳躍
- **Enter**: 確認選擇
- **滑鼠點擊**: 選擇選項

## 遊戲目標

收集愛心，回答問題，最終與 Coco 相遇！
每個愛心都會影響結局，努力獲得完美結局吧！

### 問題系統

遊戲包含 **16 個精心設計的問題**，涵蓋：
- 寶貝的喜好和夢想
- 兩人的甜蜜回憶
- 對未來的共同期待
- 彼此的第一印象和約會經歷

所有問題都經過精心設計，讓玩家更了解自己的寶貝！

### 遊戲機制

- **愛心系統**: 初始 3 顆愛心，答對問題 +1，撞到小男孩 -1，上限 12 顆愛心
- **時間壓力**: 回答問題時時間繼續倒數，增加遊戲緊張感
- **專注回答**: 問題對話框期間，小男孩和泡泡暫停移動，避免干擾
- **即時結束**: 當愛心歸零時，立即觸發 Game Over 結局
- **時間結束**: 當時間倒數到0時，會觸發 Game Over 結局，顯示 "TIME UP" 字幕
- **結局觸發**: 愛心歸零、時間結束或碰到狗都會觸發結局
- **狗狗重置**: Game Over 後重新開始時，狗狗回到初始趴著狀態

### Game Over 系統

當愛心歸零或時間結束時，會觸發特殊的 Game Over 畫面：

- **畫面效果**: 完全轉黑，中央顯示大字白色文字（"GAME OVER" 或 "TIME UP"）
- **呼吸特效**: 標題文字每 1 秒淡入淡出，像呼吸一樣
- **選單出現**: 2 秒後，下方漸漸浮現選單選項
- **逐字效果**: 選單選項會逐字打字效果出現，模擬 RPG 對話框
- **游標效果**: 當前選項會有 `>` 和 `<` 包覆，並微微放大縮放，`><` 符號具有緩慢閃爍特效（100%到40%透明度），與其他選項保持一致

#### Game Over 類型

- **GAME OVER**: 愛心歸零時顯示
- **TIME UP**: 時間結束時顯示

### 生氣結局特殊處理

當玩家得分較低（0-3顆愛心）觸發生氣結局時：

- **動畫播放**: 先播放完整的生氣結局動畫（5秒）
- **選單出現**: 動畫結束後，在動畫背景上顯示半透明的 "TRY AGAIN" 選單
- **游標效果**: 選單選項具有與對話框選項一致的放大縮放效果，`><` 符號具有緩慢閃爍特效
- **重新挑戰**: 玩家可以按 Enter 重新開始遊戲

#### 選單選項

- **> 再挑戰一次 <**: 從 START 點進去的開始畫面

#### 操作方式

- **Enter**: 確認選擇
- **滑鼠點擊**: 選擇選項

### 結局系統

- **0-3 愛心**: 低分結局（生氣動畫）+ "TRY AGAIN" 標題
- **4-8 愛心**: 蛋糕動畫結局 + "LOVE YOU" 標題  
- **9-12 愛心**: 親親動畫結局 + "LOVE YOU" 標題
- **片尾字幕**: 高分結局後會播放電影片尾字幕滾動效果，在結局開始時立即啟動，提前30秒出現，與結局流程完美同步，包含生日祝福、製作人員和溫馨致詞
- **結局收尾**: 字幕滾動完成後，自動回到主選單，讓玩家可以開始新的遊戲
- **狀態重置**: 確保所有結局相關狀態正確重置，避免干擾新遊戲
